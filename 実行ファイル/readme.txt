■操作方法

キーボード

[AorD キー]		：移動
[W キー]		：1P操作
[S キー]		：2P操作
[ENTER キー]	：同時操作
[上矢印キー]    ：ジャンプ
[SPACE キー]    ：攻撃(ブロックの後ろで)

コントローラー

[十字キー]		：移動
[LBボタン]		：1P操作
[RBボタン]		：2P操作
[Yボタン]		：同時操作
[Aボタン]    	：ジャンプ
[Bボタン]    	：攻撃(ブロックの後ろで)
━━━━━━━━━━━━━━━━━━━━
■ゲーム攻略方法

単体操作でスタミナゲージの回復を早めつつ
同時操作を積極的にしながらブロックの後ろまで行き攻撃、
とまると多くの攻撃食らうのでなるべく動き続ける。
